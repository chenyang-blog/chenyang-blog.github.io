美食日记

香锅大王

Yang Chen 发布于 2019-07-10

So — 折腾了快一个月，我的美食日记正式开工啦！欢迎大家的评论和建议 =͟͟͞͞ ٩( ๑╹ ꇴ╹)۶

身为一个爱吃辣的四川人，美食博客肿么能不从火锅/香锅开始呢！所以这第一站便是我家附近的香锅大王(Szziling Pot King)！虽然这家店因为地理位置原因已经成了我周末的固定食堂，但是几个月下来还是没有吃腻，我也有不少想记录以及和大家分享的东西。

位置

这家店在 122 S Sunnyvale Ave, Sunnyvale, CA 94086，位于Sunnyvale Downtown, 算是这一片最繁华的地区，旁边还有另一家味道不错的中餐馆以及一个煎饼果子店。Downtown中间有一片特别大的露天停车场，旁边的Target里面也有好多车位，所以停车基本不用愁。

当时选房子的时候，就是看到这三家店，再加上一个Target和2019年要新开的AMC+Whole Food，才毅然决然的决定多花点钱住在这附近。事实证明这个选择是多么明智！周末走路Shopping、吃饭、泡酒吧甚至让我有了湾区不用车也可以活的很滋润的错觉23333

这家店虽然每次去餐厅都不少人，但是因为我基本都是正点去的（5：30-7：00），所以也没怎么排过队。但是如果是周末7：30以后去的话，每次还是能看到有几个在排着队的人，总的来说不是很需要预定，座位也不是特别拥挤。

菜单

这家的菜基本是香锅 + 小菜 + 饮料。

香锅有小、中、大锅之分，大概对应1-2人 、 2-3人、3-5人的量。

香锅口味可选麻辣、香辣、蒜香和别的（其它的味道还能叫香锅么！）; 辣度有不辣、微辣、中辣、重辣，主料可选鸡丁、牛蛙、牛肉、羊肉、牛腱等等; 配料可加各种蔬菜。

小菜推荐的有猪耳、鸭脖等，其他的有待我去尝试补充。

饮料的话比较有趣的有绿豆汤、酸梅汤等。

推荐

  1. 【主菜】香锅鸡丁 - 麻辣中辣
    因为一般香锅的料很重，肉本身的味道其实差别不大，所以主料的肉质、口感就比较重要了。所以推荐一下香锅鸡丁和香锅牛蛙，吃起来肉很Q弹。

  2. 【主菜】香锅肥肠 - 麻辣中辣

  3. 【主菜】香锅兔丁 - 麻辣中辣

  4. 【小菜】鸭脖
    因为我老家附近有一个超好吃的猪耳(叫罗辣耳朵，在什邡)，所以对外面的猪耳都没有太大感觉，不过他家的鸭脖味道还是很不错的！

  5. 【饮料】绿豆汤
    正宗的绿豆汤，特别是夏天的时候喝起来特别爽！但是就是经常没有:(

评价

  1. 位置：★ ★ ★ ★

    • 不愁停车
    • 不用排队
    • 离附近的公司都近，适合周末聚餐

  2. 味道：★ ★ ★ ☆

    • 比较正宗
    • 吃的很爽
    • 第一次打分保守点

  3. 卖相：★ ★ ★

    • 看着闻着都挺有食欲的
    • 可以把人吸引进来
    • 锅里面稍微有点乱
    • 绿豆汤经常用比较low的塑料杯装

  4. 性价比：★ ★ ★ ★

    • 人均15-25能吃饱
